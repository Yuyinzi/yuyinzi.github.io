Solution

N皇后问题是一个标准的DFS问题，任意两个皇后不能出现在同一行，同一列或者对角线上(45°和135°)。针对Python，还需要考虑可变变量的问题。

对于同一行或者同一列，因为棋盘是N*N，所以只要使用一个visit数组就可以标识是否出现过。
